# Template — Premium Wins / Results

## Channel
premium-wins

## Job
Post proof, wins, results, screenshots, testimonials, or track record momentum.

## Rules
- Never invent wins.
- Never invent units, ROI, streaks, or records.
- Only use owner-provided screenshots, numbers, or text.
- If the source is unclear, ask for clarification before drafting.

## Style
- Momentum
- Proof-focused
- Confident
- Short CTA back to Premium

## Structure
1. Result headline
2. What happened, using only provided facts
3. Optional proof context
4. CTA to Premium

## Example
✅ PREMIUM RESULTS

Another clean night from the board.

Full history stays verified inside Premium.

If you’re watching from free chat, the full card is in Premium.
