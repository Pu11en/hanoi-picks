# Template — Announcements

## Channel
announcements

## Job
Public server-wide updates.

## Use for
- New offer
- Server update
- Big win recap
- Schedule update
- Important reminder

## Style
- Sales-forward
- Clear headline
- Short body
- One call to action
- Emoji-friendly, but not messy

## Required checks
- If mentioning pricing, use: $12 weekly / $30 monthly / $300 lifetime.
- If mentioning free Premium, include code HANOI.
- Never invent wins or records.

## Structure
1. Strong headline
2. What changed / what is live
3. Why it matters
4. CTA to the right channel

## Example
📢 HANOI PICKS UPDATE

Premium is live for tonight’s board.

8+ plays, full data writeups, and value spots across the slate.

If you’re still in free chat, check join-premium or use code HANOI on a partner book to unlock access.
