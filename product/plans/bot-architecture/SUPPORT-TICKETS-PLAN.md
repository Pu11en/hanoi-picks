# Hanoi Picks — Support Tickets Plan

## Decision

Support tickets are separate from the Message Operator.

The Message Operator writes official posts. Support is for members who need help.

## Live channel change

The old contact-us channel is now named support because members understand that better.

## What support is for

- Access issues
- Missing Premium role
- Payment questions
- Free Premium proof
- Partner sportsbook signup help
- Locked-channel problems
- General member issues

## Recommended setup

Use the support channel as the public doorway.

Best version:

1. Member opens support.
2. Member clicks an “Open Ticket” button.
3. A private ticket/thread opens.
4. Admins handle the issue there.
5. Ticket gets closed when resolved.

## Why not Message Operator

The Message Operator should not draft ticket posts every time. Tickets are operational support, not announcements or marketing.

## Bot/tool choice

Use a proven Discord ticket tool first instead of building custom ticket logic.

Good options to evaluate:

- Ticket Tool
- TicketsBot
- Discord private threads if the server wants to stay simple

## First version

Start simple:

- Keep support visible.
- Add a pinned instruction message.
- Add a ticket button/tool after choosing the ticket bot.
- Admins handle each issue manually.

## Suggested pinned support message

🎫 NEED HELP?

Use this channel for access issues, payment questions, and free Premium proof.

Include:
• what you’re trying to access
• whether you’re in Free Chat or Premium
• platform used if this is free Premium proof
• screenshot/proof if needed

An admin will check it and help you get sorted.
